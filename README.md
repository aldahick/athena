# athena

**WIP** Rewrite in progress, to cut MongoDB and use a bunch of new features / packages: [ES modules](https://nodejs.org/api/esm.html), [Typescript 5.0](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-0.html), [Fastify](https://npmjs.com/package/fastify), [Inversify](https://npmjs.com/package/inversify), and [Github Actions](https://github.com/features/actions). Stay tuned on the [`fastify`](https://github.com/aldahick/athena/tree/fastify) branch!

[![CircleCI](https://circleci.com/gh/aldahick/athena.svg?style=shield)](https://circleci.com/gh/aldahick/athena)
[![NPM](https://img.shields.io/npm/v/@athenajs/core)](https://npmjs.com/package/@athenajs/core)

Athena's goal is to consolidate and improve some of the best practices I've seen in codebases, and to make them reusable and extendable with minimum boilerplate in usage.

[Here's the changelog.](https://github.com/aldahick/athena/tree/master/CHANGELOG.md)
